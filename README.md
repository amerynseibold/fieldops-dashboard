# FieldOps Dashboard

A portfolio demo operations dashboard for small service businesses to track leads, quotes, jobs, follow-ups, scheduling, and pipeline visibility.

## Built With

- Next.js
- TypeScript
- Tailwind CSS
- Lucide React

## Current Features

- Operations overview dashboard
- KPI cards for pipeline value, open jobs, scheduling, and conversion
- Recent job activity
- Follow-up queue
- Live operational activity feed
- Mock data structure designed to be replaced with real database records later

## Project Goal

This project demonstrates how a small service business could replace scattered spreadsheets, text messages, and manual tracking with a clean internal operations workspace.

## Status

In progress. Current version uses mock data while the interface and workflow are being developed.
